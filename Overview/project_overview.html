MyMiniSlot: Project Visualization

Project Overview & Tech Stack

A promotional website for "MyMiniSlot," a personal slot machine device. The site showcases features, craftsmanship, gaming experience, and pre-order options.

Key Technologies:

  • HTML5
  • CSS3 (Flexbox, Grid, Animations)
  • Vanilla JavaScript
  • Vite (Dev Server & Build)
  • Google Fonts
  • Chart.js (for this viz)

Pages & Key Sections (index.html)

The main landing page (`index.html`) is structured into several key scrollable sections:

  • Header: Logo, Navigation (HOME, CRAFTSMANSHIP, EXPERIENCE, PRE-ORDER), Mobile Toggle.
  • Hero Section (#home): Main tagline, CTA, large product image.
  • Craftsmanship Section (#craftsmanship): Interactive product features with animated callouts.
  • Experience Section (#experience): Product image surrounded by game provider logos.
  • Pre-Order Section (#pre-order): Pricing, purchase info, 360 view icon.
  • Footer: Links, social media, crypto logos, copyright.

Other Informational Pages:

  • `about-us.html`
  • `cookie-policy.html` (Unique layout with large coin image)
  • `privacy-policy.html` (Layout with sidebar image)

CSS Styling & UI

Key Styling Aspects:

  • Dark Theme: Consistent dark backgrounds, light text.
  • Responsive Design: Media queries for adapting to different screen sizes.
  • Animations & Transitions: Used for navigation indicator, feature callouts, hover effects.
  • CSS Variables: For theme colors and consistent sizing (e.g., `--header-height`).
  • Modular CSS: Separate files for global styles, footer, and specific pages/sections.
